# Analytics for Retail Customer Behavior

This project analyzes retail customer shopping patterns to identify trends, customer segments, and revenue drivers.

Tools Used:
Python (Pandas, NumPy), MySQL, Power BI

Project Workflow:
Data Cleaning & Feature Engineering in Python → Transaction Modeling → SQL Analysis → Interactive Power BI Dashboard.

Key Insights:
• Clothing category generated the highest revenue
• Online sales slightly outperform offline purchases
• Discount campaigns increase transaction volume
• Customer segmentation identifies high-value and loyal customers
